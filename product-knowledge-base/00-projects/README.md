# Projects (Local Only)

Your personal workspace. Everything in this folder stays on your machine — it is never committed to Git or shared with other PMM OS users.

Use this for campaign drafts, brainstorms, client-specific work, or anything in progress. Your projects are still informed by the full knowledge base in folders `01-08`.

---

## How It Works

- This README is the only file tracked by Git — it ships with the repo so every user gets the folder
- Everything else in `00-projects/` is gitignored
- Create subfolders however you like — there's no enforced structure here
