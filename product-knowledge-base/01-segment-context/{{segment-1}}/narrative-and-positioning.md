## Overview
This document provides the strategic narrative and positioning framework for [PRODUCT NAME] targeting [SEGMENT] organizations ([GW RANGE]). It defines the **market context** (where customers currently see you), **competitive alternatives** (what they'd use/do without you), and **positioning** that differentiates [PRODUCT NAME] in the [PRODUCT CATEGORY] market.

**Key Positioning Principles Applied:**
- **Context First:** Start with understanding the market context customers see you in
- **Competitive Alternatives:** Define what customers would do/use if your product didn't exist (not just direct competitors)
- **Own One Position:** Establish one clear position you own in this segment's mind
- **Category Creation:** Create a new category if you can't win in an existing one
- **Simplify:** Focus on one core differentiator, not many features

   - Reference the primary positioning statement in all content
   - Ensure messaging aligns with the ONE position you own
   - Use positioning rationale to inform messaging approach

2. **Address Competitive Alternatives**
   - Differentiate vs. ALL alternatives, not just direct competitors
   - Reference competitive alternatives table when creating messaging
   - Use "How You're Different" points in content

3. **Leverage Unique Differentiators**
   - Lead with PRIMARY differentiator in all content
   - Support with secondary differentiators naturally
   - Ensure differentiators are unique vs. ALL alternatives

4. **Apply Market Context**
   - Understand what category segment sees you in
   - Address context gaps in messaging if needed
   - Use category strategy to inform positioning

5. **Reference Proof Points**
   - Pull quantitative and qualitative evidence naturally
   - Use customer success stories with specific metrics
   - Include proof points by message in content generation
